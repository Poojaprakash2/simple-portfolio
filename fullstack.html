Projects

IoT-based Smart Irrigation using Deep Learning Model

Designed an intelligent irrigation system using soil/moisture sensors, weather data and a lightweight deep learning model to predict irrigation needs. The system controls valves through a microcontroller and reports telemetry to a cloud dashboard. Tech: Arduino/ESP32, Python (model), MQTT, Flask backend.

Highlights: Reduced water usage by automating schedules, model-based decision making, remote monitoring.

Smart Pen for Disabled Persons

Developed an accessible pen device for users with visual/hearing impairments. It includes tactile feedback, handwriting capture, and audio output. The device bridges mechanical and electronic features to provide improved usability for special-needs learners. Tech: Embedded C, accelerometer sensor fusion, basic NLP for text-to-speech.

Highlights: Accessibility-first design, hardware prototypes, user testing with sample users.

Home Automation using DC Power Supply

Implemented a home automation prototype controlled over Wi-Fi that runs on a DC power supply setup. Includes relay-based switching for lights and fans, and a web UI for manual/automatic control. Tech: ESP8266/ESP32, Node/Flask backend, Bootstrap UI.

Highlights: Energy-efficient DC design, remote control and scheduling.
